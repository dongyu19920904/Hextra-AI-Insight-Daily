### 3. [清华开源 TurboDiffusion：视频生成提速 200 倍，5秒视频不到2秒出](https://www.bestblogs.dev/en/article/af2dec69)

### 6. [华为诺亚方舟实验室发布 SCOPE：让 Agent 学会从错误中进化](https://www.bestblogs.dev/en/article/d02a4e21)

为什么 AI Agent 总是在同一个坑里反复摔跤？华为诺亚方舟实验室和港中文的研究给出了答案：因为 Prompt 是静态的，Agent 看到错误信息也学不会。他们搞出的 SCOPE 框架，让 Prompt 能在执行过程中自我进化，从错误和成功经验中提炼规则。效果？HLE 基准上成功率从 14% 飙到 39%，GAIA 上从 33% 到 57%。这可能是 Agent 真正变"聪明"的关键一步。
